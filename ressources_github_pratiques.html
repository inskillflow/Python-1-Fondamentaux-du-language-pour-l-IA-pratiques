Comment utiliser ces ressources ?

Option 1 : Utilisation directe sur GitHub

Cliquez sur les liens ci-dessus pour voir les notebooks. Utilisez le bouton "Open in Colab" sur GitHub pour les exécuter directement dans Google Colab sans installation.

Option 2 : Cloner le dépôt complet

Si vous maîtrisez Git, clonez tout le dépôt localement pour travailler hors ligne :

git clone https://github.com/inskillflow/Python-1-Fondamentaux-du-language-pour-l-IA-pratiques.git
cd Python-1-Fondamentaux-du-language-pour-l-IA-pratiques
⚠️ Note importante : Tous les notebooks sont conçus pour être exécutés dans Google Colab. Vous pouvez également les utiliser localement avec Jupyter Notebook installé sur votre machine.
