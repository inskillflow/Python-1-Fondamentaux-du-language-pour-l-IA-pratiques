Option 1 : Utilisation directe sur GitHub

Cliquez sur les liens ci-dessus pour voir les fichiers directement. Pour les notebooks .ipynb, utilisez le bouton "Open in Colab" pour les exécuter dans Google Colab.

Option 2 : Cloner le dépôt complet

Si vous maîtrisez Git, clonez tout le dépôt localement :

git clone https://github.com/inskillflow/Python-1-Fondamentaux-du-language-pour-l-IA.git
cd Python-1-Fondamentaux-du-language-pour-l-IA
